- Color-coded health categories
- BMI scale bar with marker
- User history saved to JSON
- BMI trend chart (Matplotlib)

## How to Run
pip install matplotlib
python bmi_calculator.py

- Tkinter (GUI)
- Matplotlib (Charts)
- JSON (Data Storage)
